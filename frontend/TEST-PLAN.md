## БЛОК 3: Аналітика (/admin/dashboard)

- [ ] 4 stat cards завантажуються: Виторг сьогодні, Замовлень сьогодні, Середній чек, Активних змін
- [ ] **КЛЮЧОВА ПЕРЕВІРКА**: Виторг сьогодні = сума оплачених замовлень з POS (Блок 0)
- [ ] Замовлень сьогодні = кількість оплачених замовлень з POS (Блок 0)
- [ ] Середній чек = Виторг / Замовлень (математика)
- [ ] Активних змін = 1 (якщо зміна відкрита) або 0
